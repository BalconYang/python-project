# git单独为项目配置的方法(全局和单独配置都存在的时候会默认使用项目单独配置的)：
1. 打开项目所在目录，找到隐藏的.git文件夹。
2. 编辑器打开文件夹里的config文件。
3. 添加这三行到文件：
[user]
    name = XXX(自己的名称)
    email = XXXX(邮箱)
